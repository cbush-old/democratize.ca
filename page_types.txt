Page types descriptions:
  
  HOME PAGE
    Introduces users to Democratize.ca. 
    Showcases featured articles from all categories.
    Showcases featured bills.
    Links to contribution page
  
  ABOUT PAGE
    Further detailed explanation of Democratize
    Profiles of co-creators and contributors
    Links to contribution page
    
  BILLS HOME PAGE
    Showcases articles from the bills category
    Showcases more featured bills
  
  BILL PAGE
    Features one specific bill
    Contains links to previous/next bill in any number of threads
    Displays bill summary
    Shows the bill's categorization tags
    Displays the bill's current status in parliament
    Displays links to full text of each of the bill's editions
    Shows bill's sponsor photo, name, title, riding, and a list of his or her other bills
    Allows the user to vote yes or no on the bill
    Displays the result of public votes
    Displays a breakdown of public votes by riding 
    Displays the result of parliamentary votes (if applicable)
    Displays links to news sites with articles relating to this bill
    Allows users to comment and shows other comments on the bill
    Has social media integration buttons
    
  BILL SEARCH PAGE
    Contains a list of bills and the names of their sponsors
  
  COMMENTS HOME PAGE
    Shows where comment activity has been going on
  
  CONTRIBUTE PAGE
    Presents ways that people can contribute to the site
  
  LICENSE PAGE
    Contains the creative commons/GPL/etc license of the site and disclaimer that we're not officially related to parl.gc.ca
   
  LOGIN/REGISTER PAGE
    Username/email and password field for login
    Registration information for new users:
    - E-mail address
    - Desired username
    - Desired password
    - Repeated password
    
  MPS HOME PAGE
    Showcases articles from the MP category
    Showcases featured MPs
    
  MP PAGE
    Shows the MP's photo, name, title
    Shows the MP's current riding
    Shows the MP's various office addresses and phone numbers
    Shows the MP's e-mail address and official website url
    Shows a short bio of the MP if applicable
    Displays a list of the MP's most recent bills, if applicable
    Displays a list of the MP's committees, if applicable
    Displays a list of the MP's roles
    Displays a history of Elections in which the MP was involved
 
  MP BILLS PAGE
    Shows the MP's photo, name and riding
    Complete list of the MP's bills
    
  MP ROLES PAGE
    Shows the MP's photo, name and riding
    Complete list of the MP's roles
  
  MP COMMITTEES PAGE
    Shows the MP's photo, name and riding
    Complete list of the MP's committees
  
  MP ELECTIONS PAGE
    Shows the MP's photo, name and riding
    Complete list of the MP's elections
  
  PARTIES HOME PAGE
    Showcases featured articles from the category of parties
    Contains a list of parties and their logos
  
  PARTY PAGE
    Shows the party name
    Shows the party logo
    Contains a description of the party
    Shows a complete list of party members with photos
 
  PARTY BILLS PAGE
    Shows the party name and logo
    Has a list of bills sponsored by members of the party
  
  PARTY RIDINGS PAGE
    Shows the party name and logo
    Has a list of ridings occupied by members of the party
  
  PRIVACY POLICY PAGE
    Displays the site's privacy policy
  
  RIDINGS HOME PAGE
    Shows a map of Canada on which all the ridings are marked
    Showcases featured articles from the category of ridings
    Contains a list of ridings
  
  RIDING PAGE
    Shows a map of the riding
    Shows the riding name
    Shows the riding MP's photo and name
    Shows some info about the riding
    Displays a list of previous elections
    Shows a list of users from this riding
  
  RSS PAGE
    Contains the raw XML of the RSS feed for the site
  
  SUBJECTS PAGE
    Contains a list of subjects
  
  SUBJECT PAGE
    Contains the name of the subject
    Contains a list of bills that are categorized under this subject
    
  USER PAGE - INSIDE
    Visible to the user him or herself when logged in
    Profile picture
    User name
    Real name
    Location
    E-mail
    Demographics?
    - D.O.B.
    - Profession
    - Religious affiliation
    - Political affiliation
    Options to change the above
    Recent votes
    Recent comments
    Friends?
  
  USER PAGE - OUTSIDE
    Visible to outsiders
    Profile picture
    User name
    If user wishes to make visible:
    - Real name
    - Location
    - Age
    - Demographics, as above
    - Recent votes
    - Recent comments
    - Friends
    Option to add as friend
    
  VOTES HOME PAGE
    Shows where voting activity has taken place
    Shows bills in the context of public vote (e.g. most popular, least popular, most voted, etc)
